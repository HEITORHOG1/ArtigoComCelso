# Limites e Estrutura do Artigo IEEE Access

## LIMITE TOTAL: 8-12 PAGINAS (dupla coluna)

### Distribuicao Estimada por Secao

```
Abstract + Keywords:              ~0.5 pagina
I.   INTRODUCTION:                ~1.5 paginas
II.  RELATED WORK:                ~2.0 paginas
III. METHODOLOGY:                 ~2.5 paginas
IV.  RESULTS AND DISCUSSION:      ~3.0 paginas
V.   CONCLUSION:                  ~0.5 pagina
REFERENCES:                       ~1.0 pagina
-----------------------------------------------
TOTAL:                            ~11 paginas
```

### Regras de Espaco

- Introduction deve ser concisa e ir direto ao problema
- Related Work: cobrir apenas o necessario para posicionar o trabalho
- Methodology: detalhada o suficiente para reprodutibilidade
- Results: secao mais longa, com figuras e tabelas
- Conclusion: breve, sem repetir resultados

### Figuras e Espaco

- Cada figura single-column ocupa ~1/3 de coluna
- Cada figura double-column ocupa ~1/3 de pagina
- 6 figuras + 3 tabelas = ~2-3 paginas de espaco visual
- Manter equilibrio entre texto e elementos visuais
